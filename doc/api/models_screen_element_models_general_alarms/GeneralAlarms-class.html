GeneralAlarms

GeneralAlarms class Null safety ScreenElementModels

Offers a sorted list of general alarms for being displayed in StatusBar.

This class implements the general alarms that are displayed within the StatusBar. Note that these alarms are independent of the patient data and shown graphs and absolute values. These alarms are general notifications like O2 bottle empty etc.
